COMO JOGAR

    → Use W, A, S e D para movimentar o personagem.

    → Clique com o botão esquerdo do mouse (M1) para atirar.

    → Elimine os inimigos para ganhar pontos.

    → Cada inimigo derrotado vale +1 ponto.

    → Não encoste nos inimigos.

    → Se um inimigo tocar em você, é fim de jogo.


    OBJETIVO

    Sobreviva o máximo de tempo possível e consiga a maior pontuação!
